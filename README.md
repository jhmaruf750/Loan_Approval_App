# Loan Approval Prediction App

[![Streamlit](https://img.shields.io/badge/Streamlit-App-brightgreen)](https://loanapprovalappjhmaruf.streamlit.app/)

A **web-based machine learning application** to predict whether a loan will be approved or not based on user inputs. Built with **Streamlit, Python, and Scikit-Learn**, this app makes loan prediction fast, simple, and interactive.
